# AI-number-identifier
You will find attached a python code for a full deep learning model to classify hand written numbers (from 0 to 9 ) 
<img width="1441" height="989" alt="image" src="https://github.com/user-attachments/assets/08fcb34f-6375-445e-b39c-e458b83d216b" />
As folow we have chosen 9 number images from the mnist liberary 
We have built all the CNN layers (convolution , pooling and FC layer ) from scratch .
This project is a good starting point for those who want to understand how the neural network works and diffrent layers and processes behind image classification.
